Performance

  • performance drops when there are a significant number of $watches
  • don't show all elements at once (e.g. use pagination)
  • use one-time {{::bindings}} when applicable
  • don't use advanced filters (filters are evaluated twice in each digest cycle)
  • instead of $rootScope.$broadcast + $scope.$on it's better to use $rootScope.$emit + $rootScope.$on
  • ES7 will introduce Object.observe - it will provide a significant boost

Directives performance

  • compile phase
    • lack of $scope
    • run once
  • link phase
    • has $scope
    • run n times in ng-repeat
